# 公安备案图标替换设计

## 目标

将全站备案底栏当前使用的灰色盾牌图标替换为公安备案平台的彩色官方徽标，使外观与用户提供的参考截图一致。

## 方案

- 从公安备案平台的官方静态资源地址获取彩色徽标，并作为本地静态资源保存在 `public/`。
- `SiteFilingFooter` 继续通过 `next/image` 加载本地图标，避免网页运行时依赖公安备案平台。
- 图标保持当前 `14 × 14` 像素显示尺寸。
- 备案文字、字号、颜色、间距、链接目标、响应式布局和交互状态全部保持不变。
- 删除被替换的灰色 `public/beian-police.svg`，避免遗留无用资源。

## 验证

- 自动化测试确认公安备案项引用新的本地图标。
- 定向 ESLint 检查备案组件和测试。
- Next.js 生产构建通过。
- 推送 `main` 后确认 GitHub Actions 自动部署成功。

## 范围

本次只替换公安备案图标资源和对应引用，不修改其他界面或业务逻辑。
