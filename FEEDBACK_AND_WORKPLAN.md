## 二、关于"公式显示问题"的技术说明

公式问题分**两种**，修法完全不同：

### 情况 A：AI 回复里的公式没有渲染成数学符号

**表现**：AI 输出了 `$E = mc^2$` 或 `\frac{a}{b}` 这样的 LaTeX 代码，但页面显示的是**原始代码字符串**，没有渲染成真正的数学公式。

**原因**：前端没有安装数学公式渲染库。

**修法**：在前端加 KaTeX 或 MathJax 库（两选一，KaTeX 更轻量）。

**给 Claude Code 的指令**（后面有详细版本）。

### 情况 B：PDF 里的公式提取后丢失或变乱码

**表现**：论文 PDF 里有公式，但 AI 总结里完全没有这些公式，或者显示成乱码。

**原因**：大多数 PDF 里的公式是以**图片**形式嵌入的，文字提取工具（unpdf/pdf-parse）只能提取文字，**图片会被丢掉**。这是 PDF 格式本身的限制，不是 bug。

**短期处理**：在 AI 总结的提示词里加一句"如果论文中有重要公式无法提取，请在总结中注明并用文字描述其含义"。让用户知道是正常现象。

**长期处理**：接入视觉理解模型（Vision API）识别 PDF 里的公式图片 → 转成 LaTeX 文本。这是大工程，暂时不做。
