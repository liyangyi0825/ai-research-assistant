## 1. 项目愿景

做一个面向大学生（尤其理工科）的网页版 AI 科研助手，帮助学生更高效地完成
文献阅读、实验数据处理和学术输出。用户打开网址即可使用，无需下载安装。

**目标用户**：本科生、研究生，主要使用场景是读论文、处理实验数据、做汇报。

---

## 2. 为什么做网页版（不是桌面软件）

- 用户点链接就能用，无需安装；一条链接 / 二维码即可推广。
- 自动跨平台（手机、电脑、平板都能用）。
- 更新一次，所有人立即用上新版。
- AI 功能需要后端服务器来调用 API（API 密钥绝不能放进前端或桌面端，会被盗用），
  网页 + 后端是天然契合的架构。

| 用途 | 选型 | 选它的原因 |
|------|------|-----------|

## 4. MVP 功能范围（第一版只做这些，务必克制）

**MVP = 上传 PDF 论文 → AI 总结 → 与论文对话**

1. 用户上传一个 PDF 文件。
2. 系统提取文字内容。
3. 调用 Claude API 生成结构化中文总结（研究问题 / 方法 / 主要结论 / 创新点）。
4. 用户可针对这篇论文继续提问，AI 基于论文内容回答。

> 先把"上传 → 处理 → 展示结果"这条完整链路跑通，再加别的功能。
> 不要在第一版就做实验图像分析、PPT 生成等复杂功能。

---

## 5. 后续路线图（MVP 跑通后再逐个做）

- **阶段 2**：英文论文翻译（保留专业术语原文）、参考文献提取与导出（支持 GB/T 7714 国标、APA、IEEE）、多篇文献对比综述。
- **阶段 3**：实验数据上传（Excel/CSV）→ 自动生成柱状图/折线图 + AI 解读；实验图像的**定性**解读（描述趋势、团聚情况等）。
- **阶段 4**：SEM 等图像的**定量**分析（颗粒大小、分布直方图）——需要单独的 Python 图像处理服务；论文 → PPT 自动生成。
- **阶段 5**：用户账号、历史记录、文献库、小组协作、DDL 提醒等。

---

## 6. ⚠️ 重要技术说明（必须遵守）

**SEM 图像的定量分析不能只靠 AI 视觉模型。**
AI 模型擅长"定性描述"（如：颗粒分布不均、存在团聚），但**无法给出可靠的精确数值**
（如平均粒径 ±2nm、粒径分布直方图）。精确测量需要：比例尺标定（像素↔纳米换算）、
图像分割、逐个颗粒统计——这必须用 OpenCV / scikit-image 这类图像处理库实现
（即科研界用 ImageJ/Fiji 做的事）。

**正确架构 = 两层结合**：
- 用图像处理库做真正的分割与测量（得到可靠数字）；
- 再用 AI 负责"解读和写结论"。

产品定位上要区分清楚：是"严谨测量工具"还是"快速定性辅助"。
绝不能让 AI 直接编造测量数字给学生写进论文。

## 7. 预算（关键，分两块看）

> 价格已于 2026 年 5 月核对。AI 价格会变动，使用前请以官方页面为准。

### A. 「建站工具」费用 —— 你用 Claude Code 写代码

Claude Code 可通过 Claude 订阅或按量 API 驱动。对零基础、专注一个项目的你：
- **Claude Pro：约 $20/月** —— 入门够用，但写代码比较费额度，可能偶尔触顶。
- **Claude Max 5x：约 $100/月** —— 重度开发更从容，不容易触顶。
- 建议从 **Pro（$20/月）** 起步，不够再升级。建完站后即可降级或退订。

### B. 「网站运行」费用 —— 你的网站对外服务时

| 项目 | MVP 阶段成本 | 说明 |
|------|------------|------|
| 服务器托管（Vercel） | **$0** | 免费额度（Hobby）足够测试和小流量 |
| 数据库/存储（Supabase） | **$0** | 免费额度足够起步 |
| 域名 | **$0 ~ $15/年** | 先用免费的 xxx.vercel.app 子域名；想要自己的域名约 $10–15/年 |
| AI API（按量计费） | **几美元起** | 见下方计算 |

**AI 按量成本计算**（用 Claude Sonnet 4.6，输入 $3 / 输出 $15 每百万 token）：
- 总结一篇论文 ≈ 输入约 1.1 万 token + 输出约 1 千 token ≈ **约 $0.05 / 篇**。
- 也就是说，总结 100 篇论文大约 $5；1000 篇大约 $50。
- 开启 prompt caching 还能进一步省钱。

### C. 起步总预算

- **只为做出 MVP 并自己测试**：用 Pro 建站（$20）+ 全套免费额度 + 几美元 AI 测试费
  ≈ **$25 上下就能跑起来**。
- **开始有真实用户后**：AI 费用随用量增长，届时应给用户加使用次数限制，
  或考虑让用户自带 API key / 收费来覆盖成本。Vercel 若超免费额度，Pro 约 $20/月。
